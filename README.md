# petadoption 資策會專題 寵物送養平台

主要功能：

一、會員系統：會員註冊/登入、管理與查看送養&領養的相關資訊及申請狀態。

二、領養系統：提供使用者直接查看或設定條件篩選後呈現動物送養資訊並申請領養動物。

三、刊登送養系統：刊登動物送養資料並發佈相關資訊與條件。

四、宣導系統：科普流浪動物相關知識、列出幫助流浪動物的方法與途徑。

五、後台系統：以管理員帳號進入網站後台，進行網站會員管理&領養/送養相關資訊管理。
